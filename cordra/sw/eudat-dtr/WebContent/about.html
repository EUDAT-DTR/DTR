About EUDAT-DTR

What is it?

EUDAT-DTR is a user-friendly, secure, and trusted service that allows researchers, scientific communities and citizen scientists to define, record, query, and disseminate the descriptions (Data Type Records) of the information contained in their research datasets, as well as the assumptions and semantic contexts that need to applied to it.

Currently, EUDAT-DTR allows researchers to include the following semantic information to their Data Type Records:

  • measurement units,
  • reference coordinate systems,
  • variable names used in the data,
  • and applicable standards

By querying the Data Type Records contained in the service, researchers and automatic processes alike can discern the internal structure of a scientific dataset, parse its contents and correctly apply the semantic assumptions and contexts intended by the dataset's original creator. Moreover, the service allows to add value to research data by assigning Persistent Identifiers (PIDs) that ensure long-lasting access and references to the stored Data Type Records.

EUDAT-DTR is based on CNRI's Cordra, a Digital Object Management software that provides facilities for the creation of, and access to, digital information as discrete data structures with unique, resolvable identifiers based on the Handle System.
